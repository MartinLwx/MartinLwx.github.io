public class Hw00 {

        // Creative Exercise 1a: Drawing a Triangle
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < i + 1; j++) {

    }

public class Hw00 {
    public static int max(int[] m) {
        int maxValue = -1;
        for (int e: m) {
            if (e > maxValue) {
                maxValue = e;
            }
        }
        return maxValue;
    }
    public static void main(String[] args) {
        int[] numbers = new int[]{9, 2, 15, 2, 22, 10, 6};
        System.out.println(max(numbers));   // exercise 2
    }
}

Exercise 4

Write a function windowPosSum(int[] a, int n) that replaces each element a[i] with the sum of a[i] through a[i + n], but only if a[i] is positive valued. If there are not enough values because we reach the end of the array, we sum only as many values as we have.

The basic idea is using 2 for loops.

In the outside loop, we need to check every element in a[i]. By using continue keyword in java, we can easily skip the negative number case.

In the inner loop, our main job is to calculate the sum of a[i] through a[i + n]. We need to check j < a.length so that j is a leagal index. Although the hint says we can use break, I did this by adding j < a.length in the for loop. :)

And I use the Array.toString() method so we can print the result in a clear way.

import java.util.Arrays;

public class Hw00 {
    public static void windowPosSum(int[] a, int n) {
        for (int i = 0; i < a.length; i++) {
            if (a[i] < 0) {
                continue;
            } else {
                int windowSum = 0;
                for (int j = i; j <= i + n && j < a.length; j++) {
                    windowSum += a[j];
                }
                a[i] = windowSum;
            }
        }
    }
    public static void main(String[] args) {
        int[] a = {1, 2, -3, 4, 5, 4};      // exercise 4
        windowPosSum(a, 3);
        System.out.println(Arrays.toString(a));
    }
}
