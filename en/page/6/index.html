MartinLwx  published on 2021-12-18 included in  category Course
Lecture 01. Shell Create a new directory called missing under /tmp 1 > mkdir -p /tmp/missing 📒 -p means Create intermediate directories as required Use touch to create a new file called semester in missing. 1 2 > cd ~/tmp/missing > touch semester Write the following into that file, one line at a time: 1 2 #!/bin/sh curl --head --silent https://missing.csail.mit.edu ⚠️ Make sure you are in the tmp/missing directory start from now.
